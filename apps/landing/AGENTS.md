# IDENTITÀ

PizzaOS è una SaaS premium per pizzerie moderne.

Mood:

- Stripe
- Linear
- Apple
- Framer
- Toast.com migliorato

Non deve sembrare:

- template economico
- sito WordPress
- gestionale vecchio
- startup cheap

Pulito, premium, moderno, high trust.

Soft e realistiche.
